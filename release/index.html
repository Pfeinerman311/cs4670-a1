Programming Assignment 1: Basic Image Processing

In this assignment, we will build some basic image processing tools. In particular, you will add code to student.py to do the following:

Todos

  1. Basic appearance manipulations
    1. Brightness : Add a constant value to the image, then clip to allowable range (0,1)
    2. Contrast : Multiply intensity by a constant value, then clip to allowable range (0,1)
    3. Normalize : Make brightness and contrast changes so that the minimum intensity of the image is transformed to 0 and the maximum intensity is transformed to 1
    4. Threshold the image
  2. Implement "same" convolution of an image with a filter.
  3. Produce blurring filters:
    1. A mean filter
    2. A Gaussian filter
  4. Perform edge detection:
    1. Produce a filter computing X derivatives (w/o blurring)
    2. Produce a filter computing Y derivatives (w/o blurring)
    3. Compute the gradient magnitude by combining X and Y derivatives and optional blurring

Once you are done, running demo.ipynb should produce the output shown in demo.html

Installation instructions

For this assignment, you will need to install a number of tools including Anaconda, python3, and Jupyter.

Anaconda installation

Anaconda will come with an installation of Python and many important packages that we will use in this class.
You may install Anaconda at the following links:
  • macOS
  • Windows
  • Linux
Be sure to install a version of Anaconda that supports python3.

Jupyter installation

Now that you have conda, installation of Jupyter is fairly simple.
You will need Jupyter notebook, which you can install using conda install -c conda-forge notebook.
After installing, you can navigate to the directory in which the notebook is contained, and run the command jupyter notebook.
